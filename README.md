# BlackRoad Admin Portal

Central management dashboard for the BlackRoad OS ecosystem. Monitor products, users, API usage, and billing from one unified interface.

## Features

- **Dashboard Overview** - Real-time stats for products, users, API requests, revenue
- **Product Management** - View and manage all BlackRoad products
- **Team Management** - Invite and manage team members
- **Billing & Payments** - Unified billing across all services
- **API Key Management** - Generate and manage API keys
- **Analytics** - Deep dive into usage metrics
- **Alerts** - Real-time monitoring and notifications

## Products Integrated

- **RoadAuth** - Universal authentication
- **RoadAPI** - Unified API gateway
- **RoadBilling** - Payment processing
- **BlackRoad AI** - Sovereign AI platform
- **LangChain Studio** - AI workflow orchestration
- **RoadSupport** - AI-powered customer support

## Tech Stack

- Pure HTML/CSS/JS
- No external dependencies
- BlackRoad Design System (Golden Ratio, brand colors)

## Design System

Built with the official BlackRoad brand:
- **Hot Pink:** #FF1D6C
